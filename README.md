# Rust WASI Hello

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/yoshuawuyts/rust-wasi-hello)

An example project showing how to build an HTTP server for WASI 0.2 built in
Rust.
